# Installation

Clone the repository:

```bash
git clone https://github.com/your-username/giriraj-marketing.git
```

Navigate to project:

```bash
cd giriraj-marketing
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build production version:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
